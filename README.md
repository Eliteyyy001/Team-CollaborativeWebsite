# FreshFold POS System

This project is a team‑developed Point of Sale (POS) system created for our collaborative web development course. The system allows users to manage products, process sales, track inventory, and generate receipts in real time.

---

## Features

### Product Management
- View product list with price and stock
- Low‑stock and out‑of‑stock indicators
- Automatic stock updates after each sale

### POS Sale Processing
- Add items to cart
- Quantity validation and override warnings
- Live receipt preview (Product, Quantity, Price, Total, Date)
- Complete sale with backend processing
- Cart clearing and UI updates

### Receipt System
- Auto‑formatted receipt layout
- Right‑aligned fields for clean presentation
- Consistent horizontal line widths
- Print and download options

### Authentication
- Login and logout system for cashiers
- Session‑based access control

### Audit Logs (if enabled)
- Tracks user actions such as sales and overrides
