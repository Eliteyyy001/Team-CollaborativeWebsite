# Team Collaborative Website

## Sprint 6 - Point of Sale (POS) System

**Completed by:** Hamza Yalouli

### Overview
This repository contains the collaborative team website project.  
**Sprint 6** focuses on implementing a complete **Point of Sale (POS)** system using PHP and MySQL.

### Features Added in Sprint 6
- Main POS interface (`pos.php`)
- Product browsing and management (`products.php`)
- Sales processing (`sales.php`)
- Reports and analytics (`reports.php`)
- Audit logging system (`audit_logs.php`, `audit_helpers.php`)
- Database connection and helper functions (`dbconnect.php`)
- Login functionality (`login.php`)
- Updated index page

### Technologies Used
- PHP
- MySQL
- HTML5, CSS3, JavaScript
- Bootstrap (for responsive design)

### How to Run Locally
1. Clone or download this repository
2. Import the database schema (if provided)
3. Configure `dbconnect.php` with your MySQL credentials
4. Place the project in your web server folder (e.g., `htdocs` in XAMPP)
5. Access via `http://localhost/Team-CollaborativeWebsite`
