# FreshFold POS System

A simple Point‑of‑Sale (POS) system built for the Team Collaborative Website project.  
This system includes login authentication, product selection, cart management, receipt preview, and sale completion logic.

---

## Features

### Login System
- Username + password authentication  
- Session‑based access control  
- Redirects unauthorized users to login page  
- Logout clears session and returns to login

### POS (Make Sale)
- Product list with price, stock, and status (In Stock / Low Stock / Out of Stock)
- Add items to cart from product list or dropdown
- Quantity validation with warnings and override prompts
- Auto‑calculated totals
- Remove items or clear entire cart

### Receipt Panel
- Shows selected items, quantities, prices, and total
- Auto‑updates before sale confirmation
- Print button
- Download button (placeholder)

### Live Clock
- Displays real‑time date and time at the top of the POS screen
